# تطبيق قائمة المهام 📋

تطبيق ويب سهل الاستخدام لإدارة المهام اليومية مع حفظ البيانات محليًا في المتصفح.

## الميزات ✨

✅ **إضافة مهام جديدة** - أضف مهامك بسهولة  
✔️ **تحديد المهام المكتملة** - علّم المهام عند إكمالها  
🔍 **تصفية المهام** - اعرض جميع المهام أو فقط المكتملة أو المتبقية  
💾 **حفظ تلقائي** - المهام تُحفظ تلقائيًا في ذاكرة المتصفح  
📊 **إحصائيات** - شاهد عدد المهام الإجمالي والمكتملة والمتبقية  
🗑️ **حذف سريع** - احذف المهام الفردية أو جميع المهام دفعة واحدة  
📱 **واجهة متجاوبة** - تعمل بشكل مثالي على الهواتف والأجهزة اللوحية  
🎨 **تصميم جميل** - واجهة حديثة مع تدرجات ورسوم متحركة سلسة  

## طريقة الاستخدام 🚀

1. افتح ملف `todo-index.html` في متصفحك
2. اكتب المهمة في حقل الإدخال واضغط "إضافة" أو اضغط Enter
3. اضغط على الخانة بجانب المهمة لتحديدها كمكتملة
4. استخدم أزرار التصفية لعرض أنواع مختلفة من المهام
5. اضغط "حذف" لحذف مهمة معينة
6. استخدم "حذف المكتملة" لحذف جميع المهام المكتملة
7. استخدم "حذف الكل" لحذف جميع المهام

## التفاصيل التقنية 🛠️

### التقنيات المستخدمة
- **HTML5** - البنية الأساسية
- **CSS3** - التصميم مع التدرجات والرسوم المتحركة
- **JavaScript الخالص** - بدون مكتبات خارجية
- **LocalStorage API** - لحفظ البيانات محليًا
